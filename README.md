# Aplicacion-Web-
Aplicación web CRUD de usuarios desarrollada con Node.js y MySQL, desplegada en contenedores Docker y orquestada con Docker Compose.

Proyecto académico y de aprendizaje donde se desarrolló una aplicación web CRUD de usuarios utilizando Node.js, Express y MySQL. El proyecto se ejecuta en un entorno contenerizado con Docker y Docker Compose, permitiendo la comunicación entre servicios y la persistencia de datos. El frontend fue diseñado con Bootstrap y una estética inspirada en la cultura de los k-dramas coreanos.

Como unico detalle puesto de github no me dejo cargarlo fue la carpeta de "node-modules" no se incluye  porque se genera automáticamente al instalar las dependencias del proyecto mediante el comando npm install.
Todas las dependencias necesarias están especificadas en el archivo package.json.

Fue desarrollado en Visual Studio Code, instalando Docker, Desktop y Node.js
